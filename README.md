# StudyGroup Finder

This project follows a full-stack architecture consisting of a frontend, backend, and MySQL database working together in a decoupled system.

## Project Description
The StudyGroup Finder is a platform for students to discover, join, and manage academic study groups efficiently.

## Architecture Overview
*   **Frontend:** A modern user interface built with React and Tailwind CSS. It interacts only with the backend through API requests.
*   **Backend:** A RESTful API built with Node.js and Express.js, handling business logic and communicating with the database.
*   **Database:** A central MySQL store maintaining structured information such as users, groups, and relationships, ensuring consistency and integrity.

## Getting Started
### Prerequisites
- Node.js
- MySQL

### Setup
1. **Backend:**
   ```bash
   cd backend
   npm install
   npm run dev
   ```
2. **Frontend:**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
